Set sitemap links for website projects in Odoo!!
To provide information about particular types of content on your sites, including video and image content, you can use a sitemap. If the pages of your site are properly linked, most of your site can usually be discovered by Google. Even so, the crawling of larger or more complex sites, or more specialized files, can be improved by a sitemap. All this can be possible by using the Odoo Website Sitemap Module.
Odoo Website Sitemap allows you to set sitemap links to demonstrate website navigation. It Serves as the directory for the pages of a website accessible to users. It helps you to plan website projects with the sitemap.
